# QuickTube AI

YouTube動画の要約をGeminiで簡単に取得できるChrome拡張機能です。

## 機能

- YouTubeの動画ページでいいねボタンの左に「要約」ボタンを追加
- ボタンをクリックすると動画に被らない位置に要約パネルが表示
- 動画URLと要約リクエストテキストが自動で生成される
- ワンクリックでGeminiを新しいタブで開いてテキストを自動入力

## インストール方法（開発者モード）

1. Chromeブラウザで `chrome://extensions/` にアクセスします。
2. 右上の「デベロッパーモード」を有効にします。
3. 「パッケージ化されていない拡張機能を読み込む」をクリックします。
4. **このフォルダ（展開した `lib` フォルダ、またはリポジトリのルート）**を選択して読み込んでください。

## 使用方法

1. YouTubeの動画ページにアクセスします。
2. いいねボタンの左にある「要約」ボタンをクリックします。
3. 表示された要約タブで「Geminiで開く」をクリックすると、Geminiが立ち上がり、自動で要約プロンプトが入力されます。

## 技術仕様

- **Manifest V3** 対応
- **YouTube SPA 対応**: ページ遷移なしの動画切り替え時もボタンを維持します。
- **Gemini 連携**: APIキー不要で、Web版Geminiとのシームレスな体験を提供します。

## 注意事項

- YouTubeの動画ページ（`https://www.youtube.com/watch*`）でのみ動作します。
- Geminiを使用するにはGoogleアカウントでのログインが必要です。
